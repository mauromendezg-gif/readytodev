Plataforma de
pagos B2B
para LATAM.

Sistema de cobros recurrentes con conciliación automática y dashboard para CFOs. Diseñado para empresas medianas de LATAM que operan en múltiples países.

Listo para dev Publicado: Febrero 2026
Overview del producto

El problema

Las empresas B2B en LATAM aún manejan cobros por transferencia manual, sin visibilidad en tiempo real del estado de pagos, con alta carga operativa en el equipo de finanzas y conciliaciones que toman días enteros.

La solución

Una plataforma de cobros recurrentes que automatiza el ciclo completo: emisión de facturas, cobro por múltiples métodos de pago (SPEI, PSE, KHIPU, tarjeta), conciliación automática y dashboard de tesorería en tiempo real para el CFO.

Propuesta de valor

  • Reducción del 80% del tiempo en conciliaciones manuales
  • Cobros recurrentes automatizados con reintentos inteligentes
  • Dashboard de tesorería multi-moneda y multi-país
  • API pública para integración con ERP existentes (SAP, Oracle)
  • Cumplimiento fiscal embebido por país (CFDI, factura electrónica)
PRD — Requerimientos

Usuarios objetivo

  • CFO / Gerente financiero: Necesita visibilidad de flujo de caja en tiempo real sin depender del equipo contable.
  • Equipo de cobranza: Quiere automatizar seguimiento a clientes morosos sin emails manuales.
  • Cliente B2B pagador: Quiere pagar con su método preferido con evidencia inmediata.

Flujos principales

  • Crear y enviar factura/cobro desde el dashboard
  • Cliente recibe link de pago con múltiples métodos disponibles
  • Pago procesado → conciliación automática → notificación al CFO
  • Reintento automático ante rechazo (hasta 3 intentos con backoff)
  • Reporte mensual de tesorería exportable a Excel / PDF

Métricas de éxito

  • Tiempo de conciliación < 5 min (vs. 6–8 horas manual)
  • Tasa de cobro exitoso al primer intento > 78%
  • Días de cartera vencida reducidos en 40%
  • NPS empresas pagadoras > 50
Arquitectura técnica

Frontend

Next.js 15 con App Router. Dashboard construido con Tremor (componentes de datos). Internacionalización con next-intl para MX, CO, CL, PE. PWA para acceso móvil sin instalación.

Backend

Node.js con Fastify. Colas de trabajos con BullMQ + Redis para reintentos de cobro. Webhooks firmados para notificaciones a sistemas externos.

Integraciones de pago

  • México: SPEI vía Conekta / OpenPay, OXXO Pay
  • Colombia: PSE vía Wompi / Nequi
  • Chile: Khipu, WebPay Plus
  • Tarjeta: Stripe para internacionales

Infraestructura

PostgreSQL en Neon con Row Level Security por empresa. Redis (Upstash) para sesiones y cola de trabajos. Alojado en Vercel (frontend) + Railway (backend). Logs con Axiom.
